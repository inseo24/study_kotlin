## 1부 좋은 코드

### 1장 안정성

왜 코틀린을 사용할까? 가장 큰 이유는 안전성(safety)에 있다.

코틀린은 다양한 설계 지원을 통해 애플리케이션의 잠재적인 오류를 줄여준다. 이번 장에서는 코틀린이 어떻게 이 안전성을 위한 기능을 제공하고 올바르게 사용할 수 있는지 알아보자.

이번 장의 목적은 ‘오류가 덜 발생하는 코드를 만드는 것'에 있다.
